
# File Automation Tool

## Overview
This project is a File Automation Tool developed using Python Flask and HTML.

The application allows users to:
- Enter source folder path
- Enter destination folder path
- Generate an Excel report containing:
  - File Name
  - File Size
  - File Extension

## Technologies Used
- Python
- Flask
- OpenPyXL
- HTML
- JavaScript
